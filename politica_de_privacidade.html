Política de Privacidade

Última atualização: 14 de janeiro de 2025

Introdução

Nós respeitamos sua privacidade. Este aplicativo foi projetado para funcionar sem coletar ou compartilhar qualquer dado pessoal dos usuários. Todas as funcionalidades do aplicativo são realizadas localmente no dispositivo do usuário, sem envio ou armazenamento de dados em servidores externos.

Informações que Não Coletamos

Este aplicativo:

  • Não coleta dados pessoais: Nenhuma informação, como nome, e-mail, localização ou outros dados identificáveis, é coletada.
  • Não coleta dados técnicos: Não são armazenadas informações sobre o dispositivo, como modelo, sistema operacional ou endereço IP.

Uso da Câmera

O aplicativo solicita acesso à câmera exclusivamente para:

  • Permitir a gravação de vídeos ou captura de fotos dentro do próprio dispositivo.

Nenhuma imagem ou vídeo capturado pelo aplicativo é enviado para servidores externos ou compartilhado com terceiros. Todo o conteúdo gerado pelo usuário permanece armazenado no dispositivo e sob controle total do próprio usuário.

Compartilhamento de Informações

Este aplicativo não compartilha informações com terceiros, seja para fins comerciais, publicitários ou analíticos. O usuário tem total garantia de que seus dados não serão utilizados fora do ambiente local do dispositivo.

Proteção de Dados

Embora o aplicativo não colete dados, todas as funcionalidades locais são projetadas para operar com segurança. O acesso à câmera, por exemplo, é realizado conforme os padrões de privacidade do sistema operacional Android.

Alterações nesta Política

Podemos revisar esta Política de Privacidade para garantir que ela esteja atualizada com possíveis mudanças no aplicativo ou nos requisitos legais. Recomendamos que os usuários revisem esta política periodicamente.
